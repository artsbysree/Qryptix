🔐 Qryptix
Quantum Cryptography & Attack Simulation Platform


🌐 Live Demo: https://qryptix.vercel.app
📂 GitHub Repository: https://github.com/artsbysree/Qryptix

🚀 Overview

Qryptix is an interactive platform that demonstrates:
Classical RSA Encryption
Quantum Attacks using Shor’s Algorithm
BB84 Quantum Key Distribution
Security Comparison Dashboard
Built to raise awareness about post-quantum cybersecurity.

🏗 Architecture

Frontend: React + Tailwind + Three.js
Backend (Planned): FastAPI + Qiskit
Quantum Simulation: Qiskit

🌌 Features

Interactive encryption simulation
Visual quantum attack demonstration
BB84 key exchange visualization
Security comparison dashboard

🔮 Future Scope

Real quantum backend integration
IBM Quantum Cloud integration
Post-quantum cryptography module
